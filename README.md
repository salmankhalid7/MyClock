# Clock Project

This is a simple live clock project that displays the current date and time. The clock is styled with CSS and updates
dynamically using JavaScript.

## Demo

![Project Preview](backiee-302167.jpg)

## Features

- Displays real-time date and time
- Beautiful background image
- Responsive and centered design
- Updates every second

## Files

- `indx.html` - Main HTML file that structures the clock.
- `style.css` - Styles the clock and background.
- `Time_Live.js` - JavaScript file that updates the clock dynamically.

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/your-repo.git
   ```
2. Open `indx.html` in your browser.
